# Story 1.2: Set Up Development Environment and Build System

Status: ready-for-dev

## Story

As developer,
I want a consistent development environment with automated build and test commands,
So that I can quickly iterate on code changes across all components.

## Acceptance Criteria

1. Given** the monorepo structure from Story 1.1
2. When** I run `make setup`
3. Then** all Python dependencies are installed
4. And** pre-commit hooks are configured (black, ruff, mypy)
5. And** development database is initialized (PostgreSQL via Docker Compose)
6. And** MinIO storage is running locally (Docker Compose)
